#Uruchamianie

* Wejdź na http://nodejs.org/ i ściągnij nodejs, rozpakuj, uruchom

  ./configure

  make

  sudo make install

  npm install marked --save

* Ściągnij i zainicjuj angular-bruch-seed

* wróć do katalogu projektu, zainicjuj i odpal

  ./scripts/init.sh

  ./scripts/server.sh
